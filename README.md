# Throw-box
作業を止めずに思いついたことをすぐにメモできるツール
Throw box
=========

作業を止めずにメモを投げ込む、素早いメモツール。
Obsidian・Notion などのメインのメモツールと併用することを前提に設計されています。


【特徴】

  - Alt を押しながらマウスを画面右端に移動するとパネルが開く
  - パネルを開いた瞬間にカーソルが入力欄に
  - 指定フォルダにテキストファイルとして自動書き出し（他ツールとの連携に）
  - #タグ名 を本文に書くか、タグ欄で管理
  - 既存のメモに追記できる（追記モード）
  - ライブラリで検索・ソート・ピン留め・ロック・一括操作
  - タグごとにまとめた .txt を出力
  - ダーク / ライト テーマ


【インストール】

  Throw box Setup x.x.x.exe を実行し、指示に従ってください。

  ※ Windows SmartScreen が警告を表示する場合は
    「詳細情報」→「実行」をクリックしてください。
    コード署名証明書なしで配布しているため表示されますが、
    動作上の問題はありません。


【使い方】

  ■ メモを書く（パネル）

    Alt + マウスを画面右端へ   パネルを開く
    Ctrl + Enter               メモを保存
    タグ欄に #タグ             タグを付ける
    本文に #タグ               フォーカスを外すと自動でタグに変換

  ■ ライブラリ（整理画面）

    トレイアイコンをダブルクリックで開きます。
    検索欄で本文・タグを絞り込めます。
    選択モード中にタグをダブルクリックすると
    そのタグのメモを一括選択できます。

  ■ 自動書き出し

    設定（⚙）→「フォルダを選ぶ」で保存先を指定。
    メモを書くたびに タイトル.txt が自動で書き出されます。
    同じメモを更新すると同じファイルが上書きされます。

  ■ 自動起動

    設定（⚙）→「PC起動時に自動起動」にチェックを入れると
    Windows 起動時にバックグラウンドで自動起動します。


【データの場所】

  %APPDATA%\Throw box\brain-anchor.json      メインデータ
  %APPDATA%\Throw box\brain-anchor.json.bak  直前のバックアップ
  %APPDATA%\Throw box\images\                貼り付けた画像

  アプリをアンインストールしてもデータは残ります。
  削除する場合は上記フォルダを手動で消してください。


【動作環境】

  Windows 10 / 11
